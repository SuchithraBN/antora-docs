R0034

Purchase Order (applicable to Nymbl Suite ERP-v1.10.3.24)

  • Update

    • A new "Description" column has been added to the Item grid within the Purchase Order (LPO) Details section.

      This column’s visibility is determined by the "Show Item Description YN" setting in the Transaction Setup page against LPO. By default, the Description column displays the selected item’s name but can be manually edited by the user.

R0025

Purchase Order (applicable to ERP Lite-v0.4.4.13 and Nymbl Suite ERP-v1.7.3.21)

  • New feature

    • New status added for Partial Receive → Receive and Partail Close

      • The pending items in a partial received order can Receive and can Partail Close a purchase order.

        On Partial Close, the pending items in the LPO get canceled and will not be listed in the reference of purchase receipt. The LPO status changes from Partial Receive to Partial Close.
