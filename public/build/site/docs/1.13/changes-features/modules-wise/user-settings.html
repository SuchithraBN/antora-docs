User Settings (applicable to ERP Lite-v0.5.11.16)

  • Update

    • Removed edit functionality from the grid

    • User popup design and functionality change → new tabs added

      • For new user the division, role and location is tagged through Division, Role and Location tabs.

        • A Previous and Next button to navigate to the previous and next tabs.

        • Provide the details in each tab to navigate to the next tab.

          There is a Check All checkbox in each tab to select all divisions, roles, and locations from the corresponding tabs.

    • In edit mode also the above functionality applicable

R0012

User Settings (applicable to ERP Lite-v0.4.8.12)

  • Added new three validations under User Settings page are as follows.

Case1: hascompanylocaladmin - N (N means doesnot exists)

  1. Check if local admin exists for the logged in company

  2. A blank page will get displayed with the message "Local Admin not configured"

Case2: hascompanylocaladmin - Y (Y means local admin exists)

  1. isAdmin - N , isLocAdmin - N

    Check if the logged-in user is admin or local admin - if both N

  2. A blank page will get displayed with the message "User does not have permission to view this page"

Case3: hascompanylocaladmin - Y (local admin exists)

  1. isAdmin - Y , isLocAdmin - Y

    Check if the logged-in user is admin or local admin - if any of this is value is Y

  2. User lands to User Settings page
