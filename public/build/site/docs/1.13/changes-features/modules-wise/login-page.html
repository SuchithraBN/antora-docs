R0027

ERP API (applicable to ERP Lite-v1.0.0.650 and Nymbl Suite ERP-v1.1.7.464)

  • Update

    • Company-level location admins (LAdmin = Y) will be exempt from OTP-based authentication.

R0025

Login Page (applicable to ERP Lite-v1.7.13.61)

  • New feature

    • Instead of LOGIN button, a Generate OTP button will be displayed based on the settings done through Company master page for the selected company.

      • This button will be enabled if the Key value of OTP Authentication [ NA / USER / MANAGER ] set as MANAGER/USER through Settings tab of Company master page for the selected company.

      • If the login user is Default/Support user, then the user is considered as NA.

      • The OTP expiry time is also set through Settings tab of Company master page for the selected company where the value of the Key OTP Expiry Time in Minute set as 1 (for 1 minute).

The OTP will be sent to the email ID’s assigned to the logged-in user through User/Menu Configuration page.
