Vehicle

R0030

Vehicle (applicable to ERP Lite-v1.1.0.3)

  • New feature

    • Added new fields → Chassis No. and Salik Tag

  • Update

    • Multiple and Single selection for driver based on the Company master settings.

      • If value is Single, then single driver can only be assigned to a single vehicle.

      • If value is Multiple, then multiple drivers can be assigned to a single vehicle.

    • Multiple, Single or NA for helper based on the Company master settings

      • If value is NA then Helper field will not be displayed in the module.

      • If Single/Multiple the field will be displayed in the module with the selection of single/multi helper(s) based on settings.
