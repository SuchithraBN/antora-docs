R0019

Catalog (applicable to ERP Lite-v1.4.1.10)

  • Settings section added in General Info tab for setting KDS device and KOT printer for the product group and sub group.

  • KDS option will be enabled if Enable Kitchen Display ? option has been checked.

    The values are listed based on the toggle values (single KDS/Journey) selection made through Masters > Company > Location.

    • If single KDS option is selected, then the kitchen display devices created against the login company location through Masters > Company > Kitchen Display will get listed. In this case the Device ID will get listed.

    • If Journey option is selected, then the journey created against the login company location through Masters > Company > Kitchen Display will get listed. In this case the Journey Name will get listed.

Nymbl Suite ERP

Catalog -v0.11.3.24

  • Settings tab added for setting KDS device and KOT printer for the product group and sub group.

  • KDS option will be enabled if Enable Kitchen Display ? option has been checked.

    The values are listed based on the toggle values (single KDS/Journey) selection made through Masters > Location Related > Locations.

    • If single KDS option is selected, then the kitchen display devices created against the login company location through the Display tab of Masters > Location Related > Locations will get listed. In this case the Device ID will get listed.

    • If Journey option is selected, then the journey created against the login company location through the Display tab of Masters > Location Related > Locations will get listed. In this case the Journey Name will get listed.

R000

Nymbl Suite ERP

Catalog -v0.10.3.23

An additional feature has been introduced to synchronize another catalog detail with the chosen catalog.

  • Clicking the three dots in the top right corner of the page will list the Sync option. The user can synchronize another catalog’s details with this catalog.

  • On clicking the Sync button, a sync popup with a catalog selection field (Parent Catalog) and a Sync and Close button. The user has to select the catalog from the dropdown list and click the Sync button.
