R0020

Finance (General Ledger/ Receivable Entry / Payable Entry) (applicable to Nymbl Suite ERP-v1.5.4.21)

  • Updates

    • For the unposted documents:

      1. There should be an option to change the Main account and Sub account.

      2. The system should validate the main account, sub account and currency combination while updating the accounts in case of normal and control accounts.

      3. While updating/deleting the document, check whether matching is done for the data. If matching is done, then should not allow to update/delete the document

  • Bug fixes

    • When the posted document is copied, the new document should act as an unposted document where the item details should be editable.
